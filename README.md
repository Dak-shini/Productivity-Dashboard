Why this project exists

Many productivity tools overwhelm users with too many options.
This dashboard is intentionally designed to:
- Reduce cognitive load
- Encourage focus through prioritization
- Reward consistency without pressure

The UX is inspired by the Eisenhower Matrix and behavioral psychology.
